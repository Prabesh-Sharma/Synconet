# Project Setup Instructions

**Navigate to the Project Directory**    
   **Command:**  
   ```bash
   cd synconet/
   cd server/
   npm install
   cd ../client
   npm install
```
**Environment Variable Setup** 
```bash
  cd synconet/
  cd server/
  touch .env
  cp .env.example .env
  ```
##### you can open .env file and insert the necessary environment variables
